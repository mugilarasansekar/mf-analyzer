## ◈ Core Features At A Glance

*   ⚡ **Instant Global Search:** Real-time autocompleting lookup across thousands of AMFI schemes by name or unique numeric code.
*   📈 **Interactive NAV Trajectory:** Dynamic time-series charting powered by Chart.js featuring theme-adaptive micro-grids.
*   🔄 **Compounded Rolling Returns:** Eradicate terminal date biases with annualized $1\text{Y}$, $3\text{Y}$, $5\text{Y}$, $7\text{Y}$, and $10\text{Y}$ rolling yield curves.
*   💰 **SIP Backtest Simulator:** Test recurring monthly historical investment strategies against real point-to-point chronological milestones.
*   📉 **Advanced Risk Matrix:** Instant profiling of Maximum Historical Drawdown peaks, recovery cycle speeds, and historical Win/Loss daily distribution.
*   ⚔️ **Head-to-Head Workspace:** Normalize growth track variables linearly starting from mutual data coordinate intersections to contrast asset performance head-to-head.

## 🛠️ The Technical Blueprint

This application is built entirely on modern vanilla web technologies—no heavy frameworks, zero runtime installation tracking, and completely decentralized file architecture.

*   **Structure:** Semantic `HTML5` with structured viewport layouts.
*   **Styling:** Post-modern dark/light context tokens written entirely in native `CSS3`.
*   **Engine & Analytics:** Native Object-Oriented `JavaScript` (ES6+).
*   **Chart Rendering:** `Chart.js` (UMD bundle) with an automated `date-fns` temporal scale adapter.
*   **Data Pipeline:** Connected downstream via open-access secure JSON streams from **[mfapi.in](https://www.mfapi.in)**.
